Required: Java Runtime Environment

A Java Runtime Environment (JRE), version 1.6.0 or higher, is required. To find out what JRE version you have, open a command prompt and type in "java -version".

You can download Java for Windows from here. If you're on Linux, have a look at the official software repository of your distro.

Binary Downloads For Current Version Of DocFetcher

Note: If you have a 64-bit OS, you might have to replace an installed 64-bit Java Runtime with its 32-bit counterpart in order to make DocFetcher work. 64-bit Java is currently not supported.

For a list of changes from earlier releases, see the changelog.

Platform
 Release Notes
docfetcher_1.0.3_win32_setup.exe The Windows version runs on Windows XP or later. Windows 98 is not supported. Other Windows operating systems may also work, but have not been tested and are not officially supported.
docfetcher_1.0.3_all.deb The Linux version has been tested successfully on GNOME, Xfce and KDE.
If your OS does not support debian packages, you can use the portable version instead.
docfetcher_1.0.3_portable.zip The portable version runs on all operating systems supported by either the Windows version or the Linux version. No installation is required, just extract the contents of the archive into a folder of your choice.
