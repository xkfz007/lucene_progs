Erforderlich: Java Runtime Environment

Eine Java Runtime Environment (JRE), Version 1.6.0 oder neuer ist erforderlich. Um zu sehen, über welche JRE-Version Sie verfügen, öffnen Sie ein Befehlszeilenfenster und geben Sie "java -version" ein.

Java für Windows können Sie hier herunterladen. Wenn Sie Linux benutzen, schauen Sie in den offiziellen Software-Verzeichnissen Ihrer Distribution nach.

Binärdateien für die aktuelle DocFetcher-Version

Hinweis: Wenn Sie ein 64-bit Betriebssystem haben, müssen Sie ggf. eine auf Ihrem System installierte 64-bit Java-Laufzeitumgebung durch die 32-bit Variante ersetzen, damit DocFetcher funktioniert. 64-bit Java wird derzeit nicht unterstützt.

Für eine Liste von Änderungen zu früheren Versionen siehe Changelog.

Plattform
 Versionshinweise
docfetcher_1.0.3_win32_setup.exe Die Windows-Version läuft unter Windows XP oder neuer. Windows 98 wird nicht unterstützt. Unter anderen Windows-Betriebssysteme kann es funktionieren, sie sind aber nicht getestet und werden nicht offiziell unterstützt.
