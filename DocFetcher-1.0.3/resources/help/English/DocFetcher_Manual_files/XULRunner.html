Enabling the embedded web browser / Replacing Internet Explorer as the embedded web browser

On Linux, DocFetcher makes use of XULRunner, the piece of software best known as Mozilla Firefox's rendering engine, to display HTML content inside the preview panel. On Windows the Internet Explorer is used for this purpose. There might be two DocFetcher-specific reasons to install XULRunner:

  • You're running an older Linux distro where XULRunner is not installed by default, which is why DocFetcher's embedded web browser is not available.
  • You are a Windows user and prefer to use Firefox's rendering engine instead of the one Internet Explorer is based on.
If you fall into either category, go to the XULRunner website, click on the "release notes" link of the current release and follow the instructions given there in order to make XULRunner DocFetcher's embedded browser. After installation, you must restart DocFetcher before it works.
