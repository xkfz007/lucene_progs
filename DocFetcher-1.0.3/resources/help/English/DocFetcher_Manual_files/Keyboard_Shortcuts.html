General

Shortcut Description
Alt + F
Ctrl + F		 Focus on search box
Alt + M Focus on minimum / maximum filesize panel
Alt + T Focus on file type panel
Alt + S Focus on search scope panel
Alt + R Focus on result pages
Alt + H Hide / show filter panel
Alt + P Hide / show preview panel
F1
 Open up this manual
F11
 Open up Parser Testbox

Search scope panel

Shortcut Description
Insert Create a new index
F5 Update selected indexes
Delete Remove selected indexes
Enter Open selected folders
Shift + Insert Create a new subfolder
F2 Rename selected folders
Shift + Delete Delete selected folders
Ctrl + V Move files from clipboard into the selected folder (original files will be deleted)

Result panel

The following keys only work if the result panel has the focus (press Alt + R to focus on the result panel).
Shortcut Description
Alt + 1, 2, 3, etc. Sort result page by a specific column
A-Z
 Jump to the document whose name starts with the character pressed
Enter
 Open the selected document(s)
Ctrl + C Copy selected files / file paths into clipboard
