Preferences

You can access the preferences by clicking on the "Preferences" button on the top right. The following options are available:

Option Description
Show manual on startup Whether this manual should be opened every time the application is launched.
Close tabs after successful indexing Close tabs on the indexing dialog after successfully indexing a folder. Can be disabled for debugging.
Watch indexed folders Whether DocFetcher should listen to changes in the indexed folders (i.e. addition. deletion, modification) and update their indexes after each change.
Use OR operator as default in queries (instead of AND) Whether two or more terms in a query should be concatenated with OR or AND.
Hide program in System Tray after opening files Whether DocFetcher should be sent to the System Tray after you decide to open a document, either by doubleclicking on it or pressing the Enter key.
Highlight search terms in the preview panel Whether to highlight search terms in the text-only preview (does not apply to the HTML preview).
Clear search history on exit Whether to clear the drop-down list of previously entered queries on program exit.
Highlight color The color to use for highlighting search terms in the preview panel.
Text file extensions (default) The default for the text file extensions option on the Indexing Options dialog.
HTML file extensions (default) (See default text file extensions)
Skip files (default) The default for the file exclusion pattern used on the Indexing Options dialog.
Results per page Maximum number of result items to display on a single result page.
Global hotkey The hotkey to press to bring DocFetcher to the front.
