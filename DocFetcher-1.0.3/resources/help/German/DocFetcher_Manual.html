DocFetcher 1.0.3

Einführung

DocFetcher ist eine Open-Source-Suchanwendung: Er ermöglicht Ihnen die Volltextsuche in Dokumenten auf Ihrem Computer. - Eine Art Google für den Heimcomputer.

(Tipp: Wenn Sie dieses Dokument innerhalb von DocFetcher lesen, es aber lieber in Ihrem Webbrowser anzeigen wollen, klicken Sie auf die Schaltfläche "Seite mit externem Browser öffnen" direkt oberhalb dieses Panels.)

Indexerstellung: Klicken Sie mit rechts auf "Suchbereich" und wählen Sie "Index erstellen ..." Dann wählen Sie einen Ordner, für den ein Index erstellt werden soll. Zu Beginn sollte dies ein kleiner Ordner mit nicht zu vielen Dateien sein, z. B. 50? Nach der Auswahl erscheint ein Konfigurationsdialog. Die Standardkonfiguration sollte in Ordnung sein; klicken Sie daher einfach auf die "Start"-Schaltfläche und warten dann, bis DocFetcher die Indizierung der Dokumente beendet hat.

Alternativ können Sie einen Ordner aus Ihrem Dateimanager ziehen und im Feld "Suchbereich" fallen lassen. Ein weiterer Weg zur Indexerstellung ist der Rechtsklick auf einem Ordner in Ihrem Dateimanager. Dies ist weiter unter beschrieben, im Abschnitt "Was DocFetcher alles kann".

Suche durchführen: Geben Sie einen Suchbegriff im Textfeld über der Ergebnisanzeige (die Tabelle mit den Spaltenüberschriften) ein und drücken Sie Enter. Die Ergebnisse werden in der Ergebnisanzeige angezeigt und in mehrere Seiten aufgeteilt, wenn die Anzahl der Ergebnisse eine bestimmte Grenze übersteigt. Sie können mittels der Pfeile rechts neben dem Eingabetextfeld durch die Seiten blättern.

Filtern/Sortieren der Ergebnisse: Sie können Werte in die Felder "Minimale/Maximale Dateigröße" eingeben, um Dateien mit einer Größe außerhalb dieser Bereiche auszufiltern. Sie können außerdem Einträge in den Bereichen "Dateitypen" und "Suchbereich" (de)aktivieren, um bestimmte Dateitypen oder Dateien in bestimmten Ordnern zu filtern. Klicken Sie auf die Spaltenüberschriften der Ergebnisanzeige, um die Ergebnisse entsprechend dem gewählten Kriterium zu sortieren.

Die Vorschauanzeige: Unter (oder rechts neben) der Ergebnisanzeige finden Sie die Vorschauanzeige, welche eine Text-Anzeige mit Hervorhebung des Suchbegriffs bietet, und für HTML-Dokumente zusätzlich einen einfachen, eingebauten Web-Browser. (Es kann sein, dass Sie diesen erst installieren müssen.)

(Wenn Sie dies in DocFetcher lesen und den Anweisungen im nächsten Absatz folgen, verschwindet dieses Handbuch. Sie können es entweder mittels der "Seite im externen Browser öffnen"- oder der "?"-Schaltfläche zurückholen. )

Ein Dokument öffnen: Doppelklicken Sie auf einen der Einträge im Ergebnisfeld, wenn Sie glauben, das Gesuchte gefunden zu haben. Standardmäßig wird DocFetcher in die Systemleiste minimiert, wenn eine Datei geöffnet wurde. Dies geschieht auch, wenn Sie auf die "In die Systemleiste minimieren"-Schaltfläche des Programmfensters klicken. Zum Wiederherstellen klicken Sie auf das Icon im Systemleiste oder drücken Sie die Schnellzugriffstaste (Standardeinstellung: Strg + F8).

Index-Aktualisierungen: Wenn Dokumente im Dokumentverzeichnis hinzugefügt, entfernt oder verändert werden, müssen die entsprechenden Indizes aktualisiert werden. Dies erfolgt automatisch auf zwei Arten: (1) Wenn DocFetcher gerade läuft, erkennt er die Änderungen und aktualisiert umgehend die betroffenen Indizes. (2) Wenn DocFetcher nicht läuft, werden die Änderungen von einem kleinen Hintergrundprogramm erkannt; dann werden die betroffenen Indizes aktualisiert, sobald DocFetcher das nächste Mal startet.
Hinweis: Falls Sie die portable Version von DocFetcher benutzen und das Hintergrundprogramm automatisch beim Systemstart starten möchten, müssen Sie das Hintergrundprogramm (Daemon/Dienst) manuell zur Liste der Start-Programme Ihres Betriebssystems hinzufügen.

Indizieren sie keine Partitionen/Laufwerke: Im Gegensatz zu anderen Suchanwendungen (z. B. Google Desktop) wurde DocFetcher nicht zum Indizieren kompletter Festplatten (z. B. "C:\") entwickelt und kann explodieren, wenn Sie dies versuchen. DocFetcher erwartet, dass Sie nur das indizieren, was Sie wirklich benötigen. Dies spart sowohl Zeit als auch Speicherplatz und gibt auch wesentlich bessere Suchergebnisse.

Was DocFetcher alles kann

Portables Dokumentverzeichnis: Die portable Version von DocFetcher erlaubt es Ihnen, DocFetcher, Ihre Dokumente und die damit verbundenen Indizes in einem Ordner zu bündeln, welchen Sie dann ohne Probleme verschieben können, z. B. auf Wechseldatenträger, verschlüsselte Laufwerke oder andere Computer. Dies ist auch geeignet für die gemeinsame Benutzung eines einzelnen Dokumentverzeichnisses in einem lokalen Netzwerk (LAN), oder für einen Computer mit mehreren Betriebssystemen (z. B. Dualboot-System Linux/Windows). Einzelheiten.

Starten Sie DocFetcher durch Rechtsklick auf einen Ordner: Manchmal kann es ganz angenehm sein, eine Volltextsuche in einem Ordner durch Rechtsklick auf diesen Ordner durchzuführen, ohne irgendwelche Indexdateien zu erstellen. DocFetcher bietet eine Möglichkeit (man könnte es als "Temporär-Index" bezeichnen), die dem sehr nahe kommt. Beachten Sie, dass dies eine manuelle Installation erfordert, wenn Sie die Linux- und/oder portable Version von DocFetcher verwenden. Schauen Sie hier für weitere Einzelheiten.

Erkennung von HTML-Paaren: Paare von HTML-Dateien und zugehörigen Ordnern (z. B. "foo.htm" und "foo_Dateien") können als ein einzelnes Dokument behandelt werden. Schauen Sie hier für eine Beschreibung darüber, wie dies funktioniert, die seltenen Situationen, wo es nicht klappt, und wie man diese Situationen vermeidet.

Anpassbare Textdatei-Endungen: Sie können angeben, welche Dateien als einfacher Text behandelt werden sollen. Dieses Feature kann für Programmierer nützlich sein, die Quellcode-Dateien indizieren möchten. HTML-Dateiendungen können ebenfalls angepasst werden.

Mehrere Ordner an einem Stück indizieren: Es gibt eine Schaltfläche "Ordner zur Warteschlange hinzufügen" in der rechten oberen Ecke der Indizierungs-Konfiguration, die es erlaubt, mehrere Ordner in eine Warteschlange zu stellen, in der sie nacheinander abgearbeitet werden.

Bestimmte Dateien von der Indizierung ausschließen: Sie können bestimmte Dateien von der Indizierung ausschließen, indem Sie auf regulären Ausdrücken basierende Dateinamen-Muster eingeben. Einzelheiten.
