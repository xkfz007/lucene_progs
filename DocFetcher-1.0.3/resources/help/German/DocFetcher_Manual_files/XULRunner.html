Aktivieren des eingebetteten Web-Browsers/Ersetzen des Internet Explorers als eingebetter Web-Browser

Unter Linux benutzt DocFetcher XULRunner (von Mozilla Firefox zur Webseiten-Darstellung verwendet), um HTML-Inhalt in der Vorschauanzeige darzustellen. Unter Windows wird der Internet Explorer dafür benutzt. Es kann zwei DocFetcher-spezifische Gründe geben, XULRunner zu installieren:

  • Sie benutzen eine ältere Linux-Distribution, wo XULRunner nicht automatisch installiert wurde und daher DocFetchers eingebetteter Browser nicht verfügbar ist.
  • Sie sind ein Windows-Anwender und wollen das Firefox-Darstellungsprogramm anstelle desjenigen benutzen, auf dem der Internet Explorer basiert.
Wenn Sie in eine dieser Kategorien fallen, gehen Sie zur XULRunner-Webseite, klicken auf den Link "release notes" der aktuellen Version und folgen den dort gegebenen Instruktionen, um XULRunner zu DocFetchers eingebetteten Browser zu machen. Nach der Installation müssen Sie DocFetcher neu starten, damit die Änderung wirksam wird.
