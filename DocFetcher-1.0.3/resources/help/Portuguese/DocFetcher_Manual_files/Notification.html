Como receber notificação de novas versões

DocFetcher está hospedado no SourceForge.net , o maior repositório de software livre na web. A criação de uma conta no SourceForge.net é gratuita e permite que você possa "monitorar" os seus projetos favoritos no SourceForge.net, ou seja, para receber uma notificação de E-mail quando uma nova versão do respectivo programa é liberada.

Para receber notificação de e-mails para novas versões deste programa, primeiro se cadastre no SourceForge.net , em seguida, vá para este site e clique sobre o botão monitorar.
