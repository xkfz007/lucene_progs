Ativando o navegador web embutido/Substituir o Internet Explorer como o navegador web embutido

No Linux o DocFetcher faz uso de XULRunner, mais conhecido como renderização do Mozilla Firefox, para exibir conteúdo HTML no painel de visualização. No Windows Internet Explorer é o utilizado para este fim. Pode haver duas razões específicas para que o DocFetcher peça que você instale o XULRunner:

  • Você está executando uma versão Linux mais antiga onde o XULRunner não é instalado por padrão, e é por isso que o navegador web embutido do DocFetcher não está disponível.
  • Você é um usuário do Windows e prefere usar o Firefox em vez do motor de renderização do Internet Explorer.
Se você estiver em uma das duas categorias, vá para o site do XULRunner , e clique em "release notes" e siga as instruções dadas, a fim de tornar XULRunner DocFetcher's como o seu navegador embutido. Após a instalação, você deve reiniciar DocFetcher para que o navegador funcione.
