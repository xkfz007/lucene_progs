Preferências

Você pode acessar as preferências, clicando no botão "Preferências" no canto superior direito. As seguintes opções estão disponíveis:

Opção Descrição
Mostrar o manual na inicialização Se este manual deve ser aberto toda vez que a aplicação for iniciada.
Fechar guias (tabs) após indexação com sucesso Fechar guias na caixa de diálogo da indexação após a indexação de uma pasta. Pode ser desativado para depuração.
Monitorar Pastas Indexadas Se o DocFetcher deve monitorar mudanças nas pastas indexadas (ou seja, adição. remoçäo, alteração) e atualizar os seus índices após cada mudança.
Ocultar o programa na bandeja do sistema após a abertura arquivos Se o DocFetcher deve ser enviado para a bandeja do sistema depois que um documento for aberto, quer pelo duplo clique sobre ele ou pressionando a tecla Enter.
Destacar termos de busca no painel de visualização Se voce quer destacar buscas na pre-visualização (não se aplica à visualização de HTML).
Cor de Destaque A cor a ser usada para realçar pesquisas no painel de visualização.
Extensões de arquivo texto (default) O padrão para extensões de arquivo texto pode ser encontrado na página: Opções de Indexação
Extensões arquivo HTML (default) (Ver texto padrão extensões do arquivo)
Ignorar arquivos (padrão) O padrão para ignorar arquivos pode ser encontrado na página: Opções de Indexação
Resultados por página Número máximo de itens de resultado para ser exibido em uma única página de resultados.
Hotkey Global A hotkey a ser pressionada para trazer o DocFetcher para a frente.
