Painel de Resultados

As seguintes teclas só funcionam se o painel estiver com o foco (pressione Alt + R para focar o painel de resultados).
Atalho Descrição
Alt + 1, 2, 3, etc Classificar os resultados da página baseado numa coluna específica
A-Z
Ir para o documento cujo nome começa com o caracter pressionado
Enter
Abrir o(s) documento(s) selecionado(s)
Ctrl + C Copiar os arquivos selecionados / caminho dos arquivos para a memória

Navegação

Atalho Onde funciona Descrição
Seta para cima ou Alt + I Tamanho e tipo do arquivo e Painel de Resultados Cima
Sera para Baixo ou Alt + K Tamanho, tipo de arquivo e Painel de Resultados Para baixo
Seta para Esquerda ou Alt + J Tamanho do Arquivo e Painel de Resultados Esquerda / Página anterior
Seta para Direita ou Alt + L Tamanho do Arquivo e Painel de Resultados Direita / Próxima página
Ctrl + I Painel de Resultados Barra de Rolagem para cima
Ctrl + K Painel de Resultados Barra de Rolagem para Baixo
Ctrl + J Painel de Resultados Rolagem para Esquerda
Ctrl + L Painel de Resultados Rolagem para a Direita
